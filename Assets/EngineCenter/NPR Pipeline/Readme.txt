<概述>
	NPR Pipeline(Non-Photorealistic Rendering)。我们基于unity2019的通用管线建立了一套非真实绘制的渲染管线。在
不影响原生URP管线的前提下，加入了我们自己需要的相关功能。
	此管线流程还在持续更新中
优势如下：
	1．目前NPR流水线版本基于的是u3d 2019，模块为Universal RP 7.3.1。
	2．更加灵活的自定义和NPR相关渲染流程。
	3．自定了NPR相关的菜单按钮以及配置面板。
	4．调整后的NPR渲染流程可以容易的归纳化到中心框架中。
	5．不影响URP本身框架，不需要对其进行调整。
	6．可独立更新URP，不影响我们的NPR流水线。
注意：
	在使用我们模块前，请确保已经安装了URP模块

<日志>
ver 1.10.2.10 - 2022.01.08 - lpf
	后处理模块增加了新的Sun Shafts功能,用于模拟丁达尔效应
ver 1.9.2.10 - 2022.01.06 - lpf
	增加了胶囊体环境遮蔽功能
	取消了Lut图的生成只能在主相机渲染的设定，改为所有相机后处理前都会走一次自动生成lut图，为了更好的使用tonemapping的处理流程
	调整了NPR管线的帧调试显示节点名字，为了更加显眼
	新增了SRenderTargetHandle结构体，用于处理RT的逻辑，并对一些模块进行了相应的替换和更新。
	相机上新增Bloom Processing功能。用以支持UI辉光的功能
	将原来管线设定中的Quality里面的Camera Blur相关设定独立了出来，在外部新建了一个折叠框
	在重叠相机上增加了Clear Depth For DOP功能，即可以指定附加相机是否在渲染Depth Only Pass时，清除深度。以灵活处理一些项目的特殊需求
ver 1.6.2.6 - 2021.12.03 - lpf
	在后处理管理上增加了全局开关，用于代码控制激活和关闭后处理。
	开放了软阴影功能。
	修正了手动Lut图效果，在取消激活后，并不能还原到没有的状态。
	管线对象暴露了后处理效果黑名单，为了用户图形设定。
	再次调整了动态Lut图生成时的RT格式，采用了自动匹配当前管线是否支持HDR的设计。以修正目前某些真机上的特效黑线问题。
	开放了Render Scale参数,但限制了其最大数为1.0f.考虑该参数以后会用于进行不同机型的分辨率缩放功能.
	管线增加运行时设定的相关设计.使美术可以通过自己来决定一些运行时的管线效果参数，用来覆盖当前Asset里面的管线设定.
	开放了级联阴影,目前只开放了2级
	调整了一些面板的显示
	根据实际项目使用方式，增加新的后处理框架，替换原URP中的后处理模块
	管线中增加后处理排除名单
	管线中增加标准检测环境设定，用于标识特殊条件下的设定
ver 1.1.0.1 - 2021.8.21 - lpf
	基于各个项目组的强烈需求,增加了基于摄像机的画面模糊功能.用户可以通过摄像机开关是否模糊，质量可以在管线配置中自行设定.
	清理了一些管线中的注释代码，但目前还没有完全清理干净
